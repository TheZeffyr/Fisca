# Fisca API

Backend service for personal finance management. Track income and expenses, identify wasteful spending, save money for goals, and analyze financial statistics.

## Features

- **Income/Expense Tracking** - add transactions with categories
- **Expense Categories** - flexible transaction categorization system
- **Moneybox** - save money for specific goals/products
- **Statistics** - analytics for month, half-year, year, 3 years
- **Spending Analysis** - identify non-essential expenses for savings

## Technologies

- Python 3.13+
- FastAPI
- PostgreSQL
- SQLAlchemy
- Asyncpg
- Alembic
