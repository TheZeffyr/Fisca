# Fisca API

Бэкенд-сервис для управления личными финансами. Позволяет отслеживать доходы и расходы, выявлять лишние траты, накапливать средства на цели и анализировать финансовую статистику.

## Функциональность

-  **Учет доходов/расходов** - добавление транзакций с категориями
-  **Категории расходов** - гибкая система категоризации трат
-  **Копилка** - накопление денег на конкретные цели/продукты
-  **Статистика** - аналитика за месяц, полгода, год, 3 года
-  **Анализ трат** - выявление необязательных расходов для экономии

## Технологии

- Python 3.13+
- FastAPI
- PostgreSQL
- SQLAlchemy
- Asyncpg
- Alembic
